# ListingAdBlock

Google検索結果で表示される広告を非表示にします。  
ChromeとBraveで利用できます。  
※ デモでは効果が分かりやすいように始めに広告が表示されるようにしています。

https://user-images.githubusercontent.com/104946009/171983008-f9ce76a6-ea3c-4b12-a9ed-0b791f65120a.mov

# 導入方法

この拡張機能は現在Googleに申請中です。承認されるまでは以下の手順で手動で導入をお願いします。  

**・このサイトの右上にある「緑のCodeボタン」を押します。**

<img width="289" alt="スクリーンショット 2022-06-04 13 27 33" src="https://user-images.githubusercontent.com/104946009/171983030-b0bed743-9980-4d5d-bdaf-7996e39c56b9.png">

**・「Download ZIP」を押してファイルをダウンロードします。**

<img width="400" alt="スクリーンショット 2022-06-04 13 27 40" src="https://user-images.githubusercontent.com/104946009/171983207-ea5cd994-ccba-45a7-b975-84d863a9726d.png">

**・DLされた圧縮ファイル(ListingAdBlock-master.zip)を解凍します。**

<img width="235" alt="スクリーンショット 2022-06-04 13 28 08" src="https://user-images.githubusercontent.com/104946009/171983221-b63345ee-965d-42c0-ba4d-7309dc236196.png">

**・Chromeのメニューバーにある「ウィンドウ」から「拡張機能」を選択します。**

<img width="293" alt="スクリーンショット 2022-06-04 13 28 23" src="https://user-images.githubusercontent.com/104946009/171983254-bb17e563-a06b-43a2-8653-7e25ebf20751.png">

**・拡張機能ページの右側にある「デベロッパーモード」をOnにします。**  
**・Onにすると左側に「パッケージ化されていない拡張機能を読み込む」ボタンが表示されます。**

<img width="1437" alt="スクリーンショット 2022-06-04 13 28 35" src="https://user-images.githubusercontent.com/104946009/171983281-ec57f1d1-ee2d-4ae2-8ca6-cec94eba53ad.png">

**・「パッケージ化されていない拡張機能を読み込む」ボタンを押し、先程解凍した「ListingAdBlock-master」を選択します。**

<img width="235" alt="スクリーンショット 2022-06-04 13 28 08" src="https://user-images.githubusercontent.com/104946009/171983372-6af6961c-1311-4fed-9492-aacb59c18372.png">

**・拡張機能ページにこの拡張機能が表示されたら導入完了です。**

<img width="417" alt="スクリーンショット 2022-06-04 13 28 49" src="https://user-images.githubusercontent.com/104946009/171983427-c8d7c556-4395-404c-a6a0-76ff893f93ca.png">
